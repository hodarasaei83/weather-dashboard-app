# Weather Dashboard

A modern weather dashboard built with React, TypeScript, and Material-UI, powered by Open Weather API.

Tech Stack
React - UI framework with TypeScript
Vite - Fast build tool & dev server
Material-UI - Modern UI components
Axios - API requests to Weather API
i18n - Multi-language support
Open Weather API - Real-time weather data

API Setup
VITE_OWM_API_KEY=
API_BASE = "https://api.openweathermap.org/data/2.5"

## Quick Start

```bash
npm install
npm run dev
